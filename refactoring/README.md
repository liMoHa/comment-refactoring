# 1차 refactoring
atoms, molecules, organisms, templates에 맞춰 제공된 소스 refactoring 

# 프로젝트 구성

```
src
├── components            
│   ├── atoms		    
│   ├── molecules	
│   └── organisms		
└── pages                 
    └── Main
     	└── MainTemplate.tsx
```

# 실행방법

1. cd refactoring
2. yarn install
3. yarn dev
4. web browser에 localhost:3000 입력
